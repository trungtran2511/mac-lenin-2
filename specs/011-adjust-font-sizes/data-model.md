# Data Model: Typography Adjustments

There are no database or JSON schema changes required for this task.

All dynamic content fetched from the public JSON files (e.g. `curriculum_knowledge.json`, `quiz_questions.json`, `chapter_tool_scenarios.json`) will remain unchanged.

The only modifications are related to how these fields are rendered in React components via CSS classes:

- Chapter titles and text paragraphs.
- Quiz questions, choices, explanations, and context excerpts.
- Salary calculator labels, regions, and results.
- AI chat response texts.
