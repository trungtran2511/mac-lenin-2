# UI Contracts: Typography Sizing Scale

This contract specifies the CSS classes and text rendering standards applied to elements throughout the application.

## 1. Reading & Learning Panels

- **Chapter Description**: Must use `text-base` (16px) with `leading-relaxed`.
- **Key Point Titles**: Must use `text-base` or `text-lg` with `font-bold` or `font-semibold`.
- **Textbook Detail Body**: Must use `text-base` (16px) with `leading-relaxed` or `leading-8`. Font weight must be `font-normal` (not `font-light` to improve contrast).

## 2. Quiz Interface

- **Question Prompt**: Must use `text-lg` (18px) or `text-xl` (20px) on large screens, and `text-base` on small mobile screens.
- **Option Button Labels**: Must use `text-base` (16px) with `leading-relaxed` and adequate padding (`p-4` or `p-5`).
- **Explanation Block**: Must use `text-base` (16px) with `leading-relaxed`.
- **Textbook Excerpt (details)**: Must use `text-sm` (14px) with `leading-relaxed`.

## 3. Form Input & Details

- **Input Labels**: Must use `text-sm` (14px) or `text-base` (16px).
- **Form Descriptions & Help Text**: Must use `text-xs` (12px) or `text-sm` (14px) depending on density.
- **Surplus Formulas & Results**: Must use `text-sm` (14px) or `text-base` (16px).

## 4. AI Consultant Interface

- **AI Answer Bubble**: Must use `text-sm` (14px) or `text-base` (16px) for comfortable reading.
- **Suggested Next Step**: Must use `text-sm` (14px) or `text-xs` (12px).
